# Attributes for `react-native-alam`

Tip: Click the list icon at the to right to search for properties / alams

## Color Utils

### color

Changes the text color of an element

`color: string`

Changes:

```js
{
  color: colors
}
```

### bg

Changes the background color of an element

`color: string`

Changes:

```js
{
  backgroundColor: colors
}
```

### opacity

Changes the opacity of an element

`opacity: number`

Changes:

```js
{
  opacity: opacity
}
```

### shadow

Add a shadow to the element

Changes:

```js
{
  shadowColor: '#000000',
  [shadowOffset.width]: 0,
  [shadowOffset.height]: 2.6,
  shadowOpacity: 0.1,
  shadowRadius: 10.0
}
```

### shadow-md

Add a shadow to the element

Changes:

```js
{
  shadowColor: '#000000',
  [shadowOffset.width]: 3,
  [shadowOffset.height]: 3,
  shadowOpacity: 0.12,
  shadowRadius: 5
}
```
